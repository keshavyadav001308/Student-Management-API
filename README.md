# Student Management API(Beginner FastAPI project)

A RESTful API built using **FastAPI** and **Pydantic** to manage student records with full CRUD functionality, validation, and automatic average calculation.

 Features

- ✅ Create Student
- ✅ Get All Students
- ✅ Get Student by ID
- ✅ Update Student (PUT – Full Update)
- ✅ Partial Update (PATCH)
- ✅ Delete Student
- ✅ Automatic Average Calculation
- ✅ Field-Level Validation
- ✅ Model-Level Validation
- ✅ Duplicate ID Protection
- ✅ JSON File as Database

 Tech Stack

- Python 3.10+
- FastAPI
- Pydantic v2
- Uvicorn
- JSON (File-based database)
